FAQs

WHAT ARE COINS?

❖ Coins are used as entry fees for each match depending on the game mode selected.

❖ Coins are important parameter for deciding rank of players.

❖ In future updates, players will be able to purchase many important and exciting items from the virtual in-game-shop.

❖ Coins can be gained in the activities below:

♦ First Time Login Bonus

♦ Match Board (Coins and Pouch of Coins)

♦ Winning Bonus

♦ Daily Login Bonus

♦ Referral Bonus

♦ Social-Media Likes & Share

♦ In-Game Rewards

WHAT IS ANTI-POISON?

❖ Anti-Poison is used to save player from snake-bite if the player lands on Snakes head.

❖ Player will get following choices if they have collected “Anti-Poison” and then land on snake-head.

♦ Use: It will immediately save player from the current snake-bite.

♦ Don’t Use: Player will be moved to snake-tail, but “Anti-Poison” will be saved for the next snake-bite.

WHAT IS TOKEN?

❖ Token is a piece use to make a move on the game board (usually in the colors red, green, blue, yellow)..

WHAT I NEED TO DO IN THE GAME?

❖ You must click on the dice to roll and the token will be moved accordingly.

❖ Player who will reach box 100 first will win. (Collect coins, Avoid Snakes, Use Anti-Poisons and Climb the ladders)

DO I NEED TO ROLL '6' TO GET ON BOARD?

❖ No,you don't need to Roll '6'to get on board.

HOW TOKEN WILL MOVE AHEAD?

❖ Token will move ahead with the value on dice after rolling.

WHAT IF I ROLL '6'?

❖ Token will move ahead by '6' places and extra Dice-Roll will be given to you for rolling '6'.

WHAT IF I ROLL '6' CONSECUTIVE 3 TIMES?

❖ Token will move ahead by '6' places 3 times.

WHAT IF TOKEN LANDS AT THE BOTTOM OF ANY LADDER?

❖ Token will be moved to the top of that ladder.

WHAT IF TOKEN LANDS ON THE HEAD OF ANY SNAKE?

❖ Token will be moved down to the tail of the snake.

DO I GET EXTRA TURN FOR HITTING OPPONENT'S TOKEN?

❖ No extra turn will be given for hitting opponent's token.

WHAT IF PLAYER GETS KILLED ON OR BEFORE 20?

❖ Players Token will be placed on Home if it will take a hit on or before '20'.

WHAT IF BOTH PLAYERS LAND IN THE SAME BOX?

❖ We called this situation as “Kill”.

❖ Player who was holding the box initially will be thrown back 20-places by the opponent if opponent lands in same box.

❖ If Players gets killed before 20, then they will be sent to “Home”.

❖ After getting killed, Player lands on the box containing some item like (Snake-Head, Ladder-Bottom, Anti-Poison, Coins, Pouch of Coins, etc.) then it will be affected by those items .

❖Example 1: Red Token is in Box 30.

Green Token is in Box 26.

Green Token rolled 4 and goes in Box 30.

Now, Red token will be thrown to Box 10 and Green token will remain in Box 30.

❖Example 2: Red Token is in Box 15.

Green Token is in Box 10.

Green Token rolled 5 and goes in Box 15.

Now, Red token will be thrown to “Home” and Green token will remain in Box 15.

❖Example 3: Red Token is in Box 50.

Green Token is in Box 48.

Green Token rolled 2 and goes in Box 50.

Now, Red token will be thrown in Box 30 and Green token will remain in Box 50.

But Red token will also be affected with (Snake-Head, Ladder-Bottom,

Anti-Poison, Coins, Pouch of Coins, etc.) if Box 30 will have anything like this.
